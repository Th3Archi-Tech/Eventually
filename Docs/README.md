# Eventually Web - Gestor de Lembretes

Este é um projecto de aplicação web para gestão de lembretes, focado em simplicidade, design limpo (inspirado em Material Design) e performance, adaptado para o contexto de **Angola**. A aplicação permite criar lembretes, alternar entre temas Claro/Escuro, ver relatórios de produtividade e armazena os dados localmente no navegador.

## 🚀 Tecnologias Utilizadas

Este projecto foi construído utilizando tecnologias modernas de desenvolvimento web para garantir rapidez e facilidade de manutenção.

1.  **React (v19)**: Biblioteca JavaScript para construção da interface do utilizador baseada em componentes.
2.  **TypeScript**: Superset do JavaScript que adiciona tipagem estática, garantindo um código mais seguro.
3.  **Tailwind CSS**: Framework de CSS utilitário para estilização rápida e responsiva.
4.  **Vite**: Ferramenta de build extremamente rápida.
5.  **Lucide React**: Biblioteca de ícones leve e consistente.

## 📊 Novas Funcionalidades

*   **Relatórios de Produtividade**: Acompanhe o seu desempenho com métricas sobre tarefas concluídas, pontualidade e índice de procrastinação (tarefas em atraso).
*   **Contexto Angolano**: Formatação de datas e linguagem adaptada para `pt-AO`.

## 📂 Estrutura do Projecto

*   **index.html**: Ponto de entrada da aplicação.
*   **index.tsx**: Ficheiro raiz do React.
*   **App.tsx**: Componente principal que gere o estado global e a navegação entre Lista e Relatório.
*   **types.ts**: Definições de tipos TypeScript.
*   **services/**: Lógica de negócios (Armazenamento, Notificações).
*   **components/**: Componentes de UI (Cartão, Modal, Relatório).
